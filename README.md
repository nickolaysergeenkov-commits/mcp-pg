# MCP-PG

A Model Context Protocol (MCP) server for PostgreSQL exploration and optimization.

## Project Structure

- `mcp-pg-server/`: The core MCP server implementation.
  - `mcp_pg_server.py`: Main server script.
  - `README.md`: Setup and usage instructions.
  - `mcp_config.json.example`: Configuration template.

## Getting Started

Please see the [mcp-pg-server README](mcp-pg-server/README.md) for detailed installation and configuration instructions.
